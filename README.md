# Prototipo Pedidos Dashboard

Prototipo en HTML/CSS/JS para gestión de pedidos de bienes/servicios.

## Incluye
- Registro de pedidos (usuario, tipo, cantidad, montos)
- Flujo de estados: Registrado → Aprobado → En proceso → Listo → Entregado
- Dashboard con métricas
- Botón de **Generar entrega** con comprobante imprimible

## Ejecutar
Abrir `index.html` directamente en el navegador.
